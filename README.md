# RAG Auto Indexer

## 📌 Описание

Проект реализует простую систему базы знаний (RAG — Retrieval-Augmented Generation) с автоматическим обновлением индекса.

Система:

- загружает текстовые данные
- разбивает их на части (chunking)
- создаёт эмбеддинги
- сохраняет векторный индекс (FAISS)
- автоматически отслеживает изменения данных через hash
- обновляет индекс только при необходимости

---

## ⚙️ Возможности

- 📥 Загрузка текстовых данных
- ✂️ Разбиение на чанки
- 🧠 Генерация эмбеддингов (HuggingFace)
- 📦 Векторное хранилище (FAISS)
- 🔁 Автообновление индекса при изменении данных
- 🛡️ Проверка актуальности через hash
- 🔄 Восстановление индекса при его отсутствии
- 🔍 Поиск релевантных фрагментов
